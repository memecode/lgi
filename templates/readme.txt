Templates Readme:
-----------------

Command line way:

- Create a new folder somewhere nearby the Lgi checkout
- Copy create_project.py into it
- Drag and drop the template folder on to create_project.py
- Type in your project name...
- Open in Visual Studio and compile.


Gui way:

- Build and open Ide/LgiIde
- Project->New From Template
- Enter a project name
- Select a template to use for the platform you're on
- Configure an output folder
- Click create, close LgiIde and build with visual studio.
